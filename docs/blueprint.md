# **App Name**: SulekhaConnect

## Core Features:

- School Information Display: Display general information about Sulekha Devi Mission School such as address, contact details, and a brief history.
- News and Announcements: A section to post news, announcements, and school events.
- Calendar Integration: An interactive calendar displaying school holidays, events, and important dates.
- Image Generation: Generative AI feature to create inspirational and relevant images of school, for the 'About us' page, to provide an engaging visual experience.
- Contact Form: An interactive contact form.
- OSINT Tools Integration: Integrate OSINT tools like number lookup and profile checkup for information gathering.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF). Evokes feelings of trust, intelligence, and open communication; represents sky/air; versatile enough to be combined with various secondary colors.
- Background color: Light sky blue (#E1F5FE). A desaturated version of the primary color.
- Accent color: Turquoise (#40E0D0). Provides contrast; lies 30 degrees to the 'left' of blue.
- Headline font: 'Playfair', a modern sans-serif, for headlines; body font: 'PT Sans', a humanist sans-serif.
- Simple and clean icons to represent different sections of the app, such as calendar, news, and contact.
- A clean, well-structured layout with clear sections for different types of information.
- Subtle animations for transitions and loading states to enhance user experience.
